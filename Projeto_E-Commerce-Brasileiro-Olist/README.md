# Projeto_E-Commerce-Brasileiro-Olist
Projeto ponta-a-ponta de Engenharia e Análise de Dados, cobrindo desde a extração de dados brutos até a visualização de KPIs estratégicos com o objetivo de transformar dados de um e-commerce real (Olist) em insights acionáveis, permitindo a análise de performance de vendas, logística de entrega e satisfação do cliente.
